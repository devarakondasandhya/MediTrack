# MediTrack – Pharmacy Management System

## Project Overview

MediTrack is a web-based Pharmacy Management System designed to manage pharmacy operations such as inventory, suppliers, customers, and sales efficiently.

It is built using **HTML, CSS, and JavaScript**, with **localStorage** used for data storage on the browser side.

This project is developed for learning and demonstration purposes.

---

## Features

### Dashboard
- Central navigation for all modules
- Quick access to system features

### Inventory Management
- Add, update, and delete medicines
- Track cost price, selling price, and stock
- Automatic stock updates after sales

### Supplier Management
- Store supplier details
- Manage contact and supply information

### Customer Management
- Maintain customer records
- Track purchase history

### Billing System
- Generate bills for purchases
- Automatic total calculation
- Updates inventory after every sale

### Sales Management
- View all sales history
- Search sales by customer or date
- View detailed bill information
- Delete unwanted records

---

## Technology Stack

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage (Browser-based database)

---


---

## How It Works

1. Medicines are added into inventory with cost and selling price.
2. Sales are generated through billing functionality.
3. Inventory stock is updated automatically after each sale.
4. Sales history is stored in localStorage.
5. Reports can be generated from stored data.

---

## Data Storage

All data is stored in browser **localStorage**, including:

- Inventory data  
- Sales history  
- Customer records  
- Supplier records  

Note: Data is browser-specific and not permanent across devices.

---

## Limitations

- No backend or database integration  
- Data is not synced across devices  
- Limited scalability  
- Depends on browser storage
